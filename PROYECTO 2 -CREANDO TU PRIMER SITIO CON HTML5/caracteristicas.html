caracteristicas

algunas de las características que destacan de PHP son:

  • Para el desarrollo de aplicaciones web dinámicas que puedan acceder a la información almacenada en la base de datos.
  • Se considera un lenguaje de fácil aprendizaje debido a que durante su desarrollo se han simplificado diferentes especificaciones, como la definición de variables primitivas, este ejemplo se hace evidente cuando se utilizan arrays php.
  • El código fuente escrito en PHP no es visible para los navegadores web y los clientes porque es el servidor responsable de ejecutar el código y enviar sus resultados HTML al navegador.
  • Capacidad para conectarse a la mayoría de los motores de base de datos en uso en la actualidad, destacando su conectividad con MySQL y PostgreSQL.
  • Capacidad para usar módulos (llamados extensiones) para expandir su potencial.
  • Hay una gran cantidad de documentos en su sitio web oficial, entre los que se explican e ilustran todas las funciones del sistema en un archivo de ayuda.
